# Course Catalog 功能设计

## 1. 定位与目标
- 统一管理教育培训产品、课程包与定价策略，支持快速组合与标准化销售。
- 帮助培训顾问与教务根据学员需求构建个性化学习路径。

## 2. 核心用户
- 教研团队：产品设计、课件维护、教材迭代。
- 招生顾问：报价、套餐配置、销售资料引用。
- 教务运营：开班计划、课时包核对。
- 财务团队：检查定价、优惠政策、返佣规则。

## 3. 产品层级模型
1. **Program（项目）**：语言培训、国际课程、科研项目等顶层分类。
2. **Track（路径）**：按目标（冲刺 / 基础 / VIP 一对一）或学段划分。
3. **Module（模块）**：听说读写板块、阶段课程、专题 workshop。
4. **Package（套餐）**：课时包、模考包、服务组合。
5. **SKU（库存单元）**：具体售卖单位，包含课时数、价格、授课形式。

## 4. 功能清单
### 4.1 产品目录管理
- 新建、复制、上下架课程产品。
- 支持多语言描述、图文、宣传资料、试听视频。
- 版本管理：更新教材或课纲时记录变更说明与审核人。

### 4.2 定价与优惠策略
- 支持多币种、多地区定价。
- 优惠分类：限时折扣、团报优惠、老学员价、预存抵扣、助学金。
- 自动化价格计算：组合套餐折扣、分期付款计划、捆绑销售 ROI 分析。

### 4.3 课时与资源配置
- 为 SKU 绑定标准课时、师资要求（资质/级别）、资料包。
- 设置可用授课形式：在线直播、线下面授、录播辅导、混合式课堂。
- 管理容量：班级人数上限、开班最低人数。

### 4.4 资料关联
- 关联教案、PPT、题库、作业模版、测评卷。
- 跟踪资料使用率、版本一致性，支持区域分发。

### 4.5 报价工具集成
- 招生顾问选取课程 SKU，生成对比方案、报价单。
- 支持“主课程 + 增值服务”组合建议。
- 输出 PDF / 邮件 / 微信小程序分享链接。

### 4.6 合规与审批
- 针对新课程启用上线审批流程：内容评审、价格审批、风险评估。
- 记录资质证明（教师资格证、考试报名资质）。

## 5. 数据结构要点
- `Program`: {id, name, category, targetAudience, objectives}
- `Track`: {id, programId, level, duration, outcome}
- `Module`: {id, trackId, skills, resources, assessmentType}
- `Sku`: {id, moduleId, hours, price, deliveryMode, capacity}
- `Promotion`: {id, skuId, discountType, value, validFrom, validTo}
- `TeachingResourceLink`: {resourceId, skuId, version}

## 6. 集成需求
- 与 Training CRM 共享课程 SKU 与定价。
- 与 Scheduling 模块联动，校验师资及教室资源是否匹配。
- 与 Content & Materials 同步资料版本。
- 与财务系统同步价格表、优惠策略。

## 7. KPI 与监控
- 课程覆盖率：主力产品的 SKU 完整度。
- 资料版本一致率、教材使用率。
- SKU 成交占比、毛利率、客单价。
- 新品上线周期、审批通过率。

## 8. 迭代方向
- 面向学员的互动课程目录（筛选、收藏、申请试听）。
- AI 推荐适合课程组合（根据测评结果、兴趣、预算）。
- 动态价格优化：结合报名数据、淡旺季自动调整折扣。
- 与供应链对接（教材、教具库存），实现自动扣减。
