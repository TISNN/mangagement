# 导师信息整理文档
---

## 📋 数据字段说明

### 基础字段
- **id**: 导师唯一标识
- **name**: 导师姓名
- **avatar**: 头像URL（云存储路径）
- **tags**: 标签数组（如：博士、招生官、学屿创始人等）
- **education**: 教育背景（文本格式，包含学位、学校、证书等）
- **serviceType**: 服务类型（文本格式，如：留学申请/文书写作/留学咨询）
- **rating**: 评分（可选，范围：4.93-5.0）
- **serviceCount**: 服务学生数量（可选）
- **category**: 分类（数组或字符串，可选值：apply、subject、language）

### 分类说明
- **apply**: 留学申请相关
- **subject**: 学科辅导相关
- **language**: 语言培训相关
