# 后台导航体系设计

## 1. 设计目标

- **快速定位**：帮助顾问、文书、客服、管理员等角色在复杂模块中迅速找到业务页面。
- **语义清晰**：使用与业务直接对应的中文或常规英文名称，避免抽象比喻。
- **可扩展性**：支持一级/二级导航的增加与调整，不破坏整体结构。
- **一致体验**：保持横向与纵向导航样式统一，减少学习成本。

| 管理控制台   | 机构内部管理、权限、流程、合规、系统配置 | 系统管理 / HR / 合规 |

### 2.2 二级导航示例

#### 内部管理
- **团队成员中心**：员工档案、排班、考勤、打卡记录（详见 `internal-management/employee-and-scheduling.md`）
- **系统设置**：站点配置、通知策略、集成管理（详见 `internal-management/system-settings.md`）
- **架构调整说明**：原“组织与权限”“流程中心”“合规监督”子模块已下线，相关流程合并至团队成员管理与系统设置，由运营策略组统一维护。

#### 客户与销售
- **线索总览**：线索看板、销售漏斗、渠道表现
- **线索详情**：线索列表、沟通记录、提醒计划
- **客户档案**：客户分群、合作状态、续约提醒
- **合同与授权**：合同库、签约流程、价格方案
- **服务中心**：B 端服务菜单、订单进度、顾问分配

#### 教育服务
- **服务总览**：学生服务概览、指标卡、AI 洞察
- **学生管理**：学生列表、档案、服务记录
- **导师管理**：导师列表、排班、任务指派
- **选校规划**：AI 推荐、人工筛选、协作会议
- **服务进度**：项目时间线、任务看板、风险预警
- **文书与申请**：文书协同、提交进度、材料管理

#### 数据分析
- **关键指标**：业绩仪表盘、项目收入、顾问 KPI
- **招生分析**：渠道分析、转化率、地区分布
- **服务分析**：服务满意度、完成率、教学质量
- **预测洞察**：招生预测、课程情况预估、资源利用率

#### 知识与文件
- **知识库**：操作手册、案例库、培训资料
- **项目复盘**：会议纪要、执行复盘、经验总结
- **模板中心**：合同模板、文书模板、设计素材
- **文件管理**：上传下载、版本控制、授权设置

## 3. 交互与布局细节

### 3.1 侧边栏结构
- 一级模块默认展开当前所在模块，其余折叠。
- 支持“迷你侧栏”：收起后仅显示图标，悬停时展开二级菜单。
- 二级菜单支持三级标签或在主区域使用页内标签切换。

### 3.2 顶部横幅
- **LOGO 与系统名称**：突出品牌及系统定位。
- **全局搜索**：支持搜索学生、客户、任务、页面等。
- **通知中心**：展示待办提醒、审批请求、系统公告。
- **用户中心**：展示角色、组织切换、个人设置及退出。

### 3.3 快捷入口区
- 位于侧栏底部，展示常用页面或收藏功能。
- 支持拖拽排序、固定/取消固定，自定义常用入口。
- 面向新用户提供默认推荐，例如“学生管理”“线索总览”。

### 3.4 面包屑与标签
- 二级页面顶部使用面包屑提示层级路径（例如 `教育服务 / 学生管理 / 学生详情`）。
- 页面内部如有多个子视图使用标签页呈现，保持信息层级清晰。

### 3.5 搜索与命令面板
- `⌘ + K` / `Ctrl + K` 打开命令面板，支持模糊搜索中文或常规英文名称。
- 搜索结果按类型分组：页面、学生、客户、任务、文档。
- 提供最近使用记录与常用功能建议。

## 4. 视觉风格建议

- **色彩**：基于品牌主色，推荐灰白 + 品牌色强调；支持浅色/深色模式。
- **侧边栏**：使用统一背景色，当前模块高亮，悬停反馈清晰。
- **图标**：统一使用简洁线性图标，与模块含义一致。
- **字体**：系统默认字体，标题加粗突出层级，正文保持易读行高。

## 5. 响应式与角色适配

- **平板/移动端**：侧边栏收纳为抽屉或底部导航；关键页面提供简化视图。
- **角色视角**：可根据角色（顾问、运营、管理员）切换导航显示范围。
- **多语言**：支持中英文切换，名称保持直译或对应业务术语。

## 6. 数据与埋点

- 记录各模块访问频次、停留时长、跳出率。
- 统计命令面板搜索词与点击率，用于优化命名或快捷方式。
- 收集快捷入口使用情况，动态调整系统默认推荐。
- 监控菜单展开/收起行为，评估信息结构合理性。

## 7. 迭代路线建议

1. **基础结构上线**：完成导航重命名、分组与样式统一。
2. **角色视角**：按角色维护导航配置，提供视角切换。
3. **命令面板升级**：引入全局搜索、最近访问记录。
4. **数据分析**：上线导航使用数据仪表盘，指导后续优化。
5. **体验优化**：根据反馈优化快捷入口、移动端适配。
