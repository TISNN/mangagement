# 项目展示大厅（Project Marketplace Gallery）

> 目标：为顾问和学生提供一个可快速搜索、对比、收藏项目的可视化大厅，帮助他们像逛超市一样挑选最合适的留学项目。

## 1. 模块定位与价值
- **顾问**：集中查看所有项目，创建推荐清单，提高接待效率。
- **学生/家长**：直观了解项目亮点、价格、服务流程，减少信息不对称。
- **运营**：通过推荐位和专题策划，增强热门项目曝光。

## 2. 用户画像与需求
| 用户 | 核心目标 | 痛点 | 大厅提供的价值 |
|------|----------|------|----------------|
| 平台顾问 | 快速找到匹配学生需求的项目 | 资料分散、难比较 | 强大筛选、收藏、清单分享 |
| 学生/家长 | 了解项目是否适合自己 | 信息不透明、判断难 | 清晰标签、真实评价、申请引导 |
| 新合作机构 | 获得曝光、被顾问理解 | 沟通成本高 | 统一展示品牌与项目优势 |

## 3. 信息架构
```
项目展示大厅
├─ 顶部检索区
│  ├ 关键字搜索
│  ├ 筛选条件（国家、阶段、预算、语种、项目类型、开课时间、客户口碑）
│  └ 快速筛选（精选项目 / 新上架 / 高口碑）
├─ 推荐区域
│  ├ 运营位（Banner、专题卡）
│  ├ 智能推荐（基于顾问最近服务学生）
├─ 项目列表区
│  ├ 卡片展示（封面图、关键数据）
│  ├ 列表视图（适合顾问批量勾选）
│  └ 对比功能（多选项目对比表）
├─ 项目详情页
│  ├ 概览（价格、周期、成功案例）
│  ├ 服务流程与交付物
│  ├ 导师团队介绍
│  ├ 学生条件与需备材料
│  ├ 支持的套餐对比
│  ├ 评价与问答
│  └ 行动按钮（收藏、加购清单、预约顾问）
└─ 我的清单
   ├ 顾问清单（可生成 PDF / 分享链接）
   ├ 学生收藏夹
   └ 清单备注与状态跟踪
```

## 4. 关键交互
- **搜索与筛选**
  - 输入关键词支持模糊匹配项目名称、国家、特色标签。
  - 筛选条件含“预算区间滑条”“开课时间（日历）”“服务语言”。
  - 支持保存筛选条件为“我的视图”。
- **项目卡片**
  - 显示：国家标记、阶段、评分、价格区间、服务时长、标签（奖学金/紧急报名/热门）。
  - 悬停动作：`收藏`、`加入清单`、`快速分享（复制链接/生成小卡片）`。
  - 若项目来自指定机构，展示机构 Logo 与认证标识。
- **对比功能**
  - 顾问可多选项目，生成对比表格（价格、周期、服务项、成功案例、退费策略）。
  - 可一键导出 PDF，供学生线下查看。
- **详情页行动**
  - `预约顾问讲解`：触发 CRM 线索或内部任务。
  - `发起签约`：直接进入合同流程（预填项目信息）。
  - `同类推荐`：展示同国家/同阶段的备选项目。

## 5. 数据来源与缓存策略
- 项目信息：来自 `project_market_item`、`project_package`、`project_feedback`。
- 筛选项：国家、阶段、项目标签等从 `service_types`、自定义标签表中获取。
- 推荐数据：结合顾问近期服务学生的画像（教育阶段、预算、目的国家）。
- 缓存策略：热门筛选结果使用 Supabase Edge Functions + Redis 缓存；详情页静态化 5 分钟更新，保证访问速度。

## 6. 与其他模块的联动
- **顾问 CRM**：顾问对项目的收藏、清单、分享行为，写入 CRM 作为学生沟通记录。
- **服务交付**：一旦签约，自动在 `student_services` 中标注来源项目 ID，方便追踪项目效果。
- **审核后台**：已下架项目在列表中实时隐藏，详情页提示“项目已暂停报名”。
- **运营工具**：专题位/推荐位配置直接来自运营控制台。

## 7. 指标监控
- 浏览量（PV）与独立访客数（UV），按国家/阶段分布。
- 搜索使用率与热门搜索词，监测顾问需求趋势。
- 收藏率、加入清单率、分享率。
- 项目详情页 → 咨询按钮点击率 → 成交率的转化漏斗。
- 顾问平均创建清单数、学生清单反馈。
- 退费/投诉的项目曝光次数占比，防止高投诉项目继续曝光。

告警规则：
- 某项目在 24 小时内投诉量激增 → 暂停推荐位展示。
- 某筛选条件零结果率持续超 7 天 → 提醒运营补充项目。
- 清单导出失败率 > 5% → 检查文档生成服务。

## 8. 可视化与 UI 建议
- **色彩**：使用暖色系强调“热门”、“即将截止”；冷色系表示“稳定”、“常规”。
- **卡片布局**：采用栅格布局，每行 3-4 张卡片；在窄屏设备下切换为两列/单列。
- **骨架屏**：加载时显示卡片骨架，保证体验顺滑。
- **无结果页面**：提供“调整筛选条件”“预约顾问协助”引导。
- **辅助说明**：在关键信息旁加入“问号提示”，解释价格构成、退费规则等。

## 9. 可扩展路线
| 阶段 | 能力 | 说明 |
|------|------|------|
| MVP | 搜索 + 筛选 + 列表 + 详情 + 收藏 | 保证顾问日常筛选流程可用 |
| V1.1 | 清单分享 + 对比导出 + 推荐位 | 支撑顾问与学生沟通 |
| V1.2 | AI 推荐 + 智能排序 | 根据学生画像推荐最匹配项目 |
| V1.3 | 项目评价体系 + 问答社区 | 提升真实性，形成口碑循环 |
| V1.4 | 外部展示模式 | 可对外开放部分项目，助力品牌曝光 |

## 10. 初学者易懂的说明
- 可以把项目大厅理解成“课程商店”，各种项目变成一张张商品卡片。
- 顾问就像导购，可以把喜欢的商品拉到一个购物篮（清单），再给学生推荐。
- 学生和家长也能自己逛，挑到合适的项目后点“找顾问”就能获得帮助。
