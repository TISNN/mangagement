# 数据导入工具

这个目录包含了用于导入各种数据到系统的工具脚本。

## QS世界大学排名导入

该工具可以将最新的QS世界大学排名数据导入到学校数据库中。

### 使用方法

1. 确保项目根目录下存在`2025 QS World University Rankings 2.2 (For qs.com).csv`文件
2. 安装依赖
```bash
npm install
```
3. 运行导入命令
```bash
npm run import
```

这个命令会自动完成：
1. 复制CSV文件到工具目录
2. 导入数据到Supabase数据库

### 单独执行步骤

如果需要单独执行某个步骤，可以使用以下命令：

- 只复制CSV文件：`npm run copy-csv`
- 只执行导入操作：`npm run import-qs`

## 排障

如果导入过程中出现错误，请检查：

1. CSV文件是否存在于正确位置
2. `.env`文件是否包含正确的Supabase连接信息
3. 确保已安装所有依赖

如果仍然存在问题，请查看详细的错误日志并根据提示解决。
