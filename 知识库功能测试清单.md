# 知识库功能测试清单 ✅

## 📋 更多选项菜单（完整功能）

点击资源卡片右上角的 **⋮ 三点按钮**，应该看到：

```
┌─────────────────────────┐
│ 👁️  查看详情            │
│ ✏️  编辑资源            │  
│ ⭐ 设为精选            │  ← 未精选时显示
│ ⭐ 取消精选            │  ← 已精选时显示
├─────────────────────────┤
│ 🔗 复制链接            │
│ 📦 归档                │
├─────────────────────────┤
│ 🗑️  删除资源           │  ← 红色，底部
└─────────────────────────┘
```

### ✅ 验证删除功能

1. **找到任意资源卡片**
2. **点击右上角的 ⋮ (三点按钮)**
3. **菜单应该展开显示**
4. **滚动到菜单底部**
5. **应该看到红色的"🗑️ 删除资源"选项**

### 如果看不到删除选项

可能的原因：
- 菜单被遮挡（尝试调整浏览器窗口大小）
- 菜单太高（滚动查看底部）
- 缓存问题（强制刷新：Ctrl+Shift+R）

## 🎨 视觉验证

### 资源卡片右上角应该有 3 个按钮：

```
┌──────────────────┐
│  [📖] 文档       │
│                  │
│  [🔖][📤][⋮]   │  ← 这里！
└──────────────────┘
```

- **🔖 书签** - 灰色/蓝色（已收藏）
- **📤 分享** - 灰色，悬停变绿色
- **⋮ 三点** - 灰色，悬停变深色

**分享菜单**：
- 白色/深色背景

- 4 个选项

**更多选项菜单**：

- 7-8 个选项
- 带分割线
- 删除选项是红色
