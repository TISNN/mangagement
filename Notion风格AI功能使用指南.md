## 🎯 如何使用

### 触发 AI 助手

**方式 1：空行按空格** ⭐（Notion 风格）
```
1. 在编辑器中新起一行（空行）
2. 按下空格键
3. AI 对话框自动弹出！
```

### AI 对话框功能

#### 1. 快速选择
点击预设选项：
- **随心创作任何内容的初稿** - AI 写一篇文章
- **起草提纲** - AI 生成大纲
- **制作表格** - AI 创建表格
- **制作流程图** - AI 画流程图
- **随心写作** - 自由创作
- **继续写作** - 续写当前内容
- **头脑风暴** - 提供创意想法
- **帮助编写代码** - 生成代码

#### 2. 自定义提问
在搜索框输入：
```
"帮我写一份会议总结"
"如何提升团队效率"
"分析一下这个项目的风险"
```
然后点击发送 ↑

---

## 📋 AI 结果操作

### AI 生成后的选项

1. **接受并插入** ✅
   - 内容插入到光标位置
   - 可以继续编辑

2. **拒绝** ❌
   - 关闭对话框
   - 不插入内容

3. **重新生成** 🔄
   - AI 再生成一次
   - 换个角度思考

## ⚙️ 配置 AI API

### 步骤 1：获取 API Key

**推荐：通义千问**（阿里云，便宜）
1. 访问：https://dashscope.aliyun.com/
2. 注册/登录
3. 创建 API Key
4. 每月有免费额度

**或使用 OpenAI**
1. 访问：https://platform.openai.com/
2. 创建 API Key

### 步骤 2：配置环境变量

在项目根目录创建 `.env` 文件：

```bash
# 通义千问
VITE_AI_API_KEY=sk-xxxxxxxxxxxxx

# 或 OpenAI
# VITE_AI_API_KEY=sk-xxxxxxxxxxxxx
```

### 步骤 3：重启服务器

```bash
# 停止服务器
Ctrl + C

# 重新启动
npm run dev
```

## ⚠️ 注意事项

### 1. API Key 安全
- 不要提交到 Git
- .env 已在 .gitignore 中
- 定期更换 API Key

### 2. 成本控制
- 通义千问：约 ¥0.002/1K tokens
- 每次 AI 调用约 ¥0.01-0.05
- 月成本约 ¥10-50（轻度使用）

### 3. 内容审核
- AI 生成的内容需要人工审核
- 不要盲目接受所有 AI 建议
- 特别是重要文档
